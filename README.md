# Sistema de Gerenciamento – Loja de Manutenção (Java)

## 📌 Descrição
Sistema desenvolvido em Java como avaliação da disciplina de Programação no curso Técnico em TI (2024).

A aplicação simula o gerenciamento básico de uma loja de manutenção de produtos, permitindo controle completo via terminal.

## 🚀 Funcionalidades
- Cadastro de produtos
- Registro de conserto com valor
- Entrega de produto ao cliente
- Relatórios:
  - Completo
  - Produtos não consertados
  - Produtos consertados e não entregues
- Cálculo de faturamento total

## 🛠 Conceitos aplicados
- Programação estruturada
- Métodos
- Vetores
- Estruturas condicionais
- Laços de repetição
- Controle de estado (boolean)
- Formatação manual de saída

## 🎯 Objetivo
Aplicar fundamentos de lógica de programação na construção de um sistema funcional que simula um problema real.
